# Tetris-2048 Hybrid 🎮

A hybrid puzzle game blending the spatial mechanics of **Tetris** with the numerical merging logic of **2048**.

## 🚀 Quick Start

Clone the repository and run the game:

```bash
# Clone the repo
git clone [https://github.com/selimsrky/Tetris-2048-Hybrid.git](https://github.com/selimsrky/Tetris-2048-Hybrid.git)
cd Tetris-2048-Hybrid

# Install dependencies
pip install numpy

# Run the game
python main.py
